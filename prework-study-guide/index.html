HTML

  • The head element contains information about the webpage for the browser.
  • The body element represents the visible content shown to the user.
  • The body element is the parent element, everything inside it are its child elements.
  • An HTML attribute defines additional information about an element.
  • "id" attribute identifies elements to select them
  • An element is a type of content on a webpage
  • The name of semantic HTML elements describe its content and the role it plays on the page.
  • span tag creates an inline division.
  • h1 > h6
  • ol is numbered list, li is bullet list

CSS

  • A margin indicates how much space we want around the OUTSIDE of an element.
  • A padding indicates how much space we want around the content INSIDE an element.
  • Increasing padding makes the entire element bigger!
  • Declarations have 2 main parts: CSS property and the value of that property.
  • Select a class with a dot .class
  • 3 ways to implement: Inline, Internal Style Sheet, External Style Sheet
  • Inline is good for quick and permanent changes.
  • Internal is embedded with style tag in HTML head (only applies to that page).
  • External file is linked in HTML head.
  • Displaying an image as a block will center it.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stage your working changes
  • git commit -m "description of changes": commits changes
  • git pull origin main: see if current branch is up to date with main (sync local branch with remote branch)
  • git push origin current-branch-name: pushes up changes
  • After merge complete, update local main branch
  • git checkout main: switches to main branch
  • git pull origin main: pull the remote main branch into the local main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS is read top to bottom.
  • JS is linked in body element of HTML.
  • If statements tell the program to leave normal flow if certain conditions are met.
  • Checks to see if condition evaluates to true or false, which determines the next action.
  • Strict equality === sees if 2 values are equal, and returns a boolean true/false.
  • An array is a single variable that holds a group of related data.
  • Arrays are used to perform tasks on several pieces of data at once.
  • Use the index number of a data piece to call it in an array (starts at 0).
  • For loops execute code of block over and over.
  • For loops have 3 components: initalization (declare a variable), condition, icrement(++).
  • The loop will run as long as the condition is true.
  • (++) Every time the code loops, the variable will increase by 1.
  • Functions are used to define and reuse blocks of code.
  • After defined, function has to be called to execute. It is not automatic like for loops or conditional statements.
