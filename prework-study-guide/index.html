Prework Study Guide

HTML

  • The head element contains information about the webpage for the browser.
  • The body element represents the visible content shown to the user.
  • The body element is the parent element, everything inside it are its child elements.
  • An HTML attribute defines additional information about an element.
  • "id" attribute identifies elements to select them
  • An element is a type of content on a webpage
  • The name of semantic HTML elements describe its content and the role it plays on the page.
  • span tag creates an inline division.
  • h1 > h6
  • ol is numbered list, li is bullet list

CSS

  • A margin indicates how much space we want around the OUTSIDE of an element.
  • A padding indicates how much space we want around the content INSIDE an element.
  • Increasing padding makes the entire element bigger!
  • Declarations have 2 main parts: CSS property and the value of that property.
  • Select a class with a dot .class
  • 3 ways to implement: Inline, Internal Style Sheet, External Style Sheet
  • Inline is good for quick and permanent changes.
  • Internal is embedded with style tag in HTML head (only applies to that page).
  • External file is linked in HTML head.
  • Displaying an image as a block will center it.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
